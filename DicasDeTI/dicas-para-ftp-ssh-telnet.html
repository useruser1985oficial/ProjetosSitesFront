Dicas para FTP, SSH e Telnet

Veja algumas dicas para conexões com FTP, SSH e Telnet. Alguns precisam ser instalados ou ativados no sistema operacional para serem usados. No Windows é só ir em Ativar ou Desativar Recursos do Windows e marcar os serviços escolhidos. No Linux e sistemas Unix eles já costumam vir instalados.

FTP

FTP é um protocolo para transferir arquivos. Basicamente fazemos assim para acessar um servidor (podendo usar o nome ou o IP):


ftp nomeouipdoservidor.com

Depois coloque o usuário e senha do servidor.

Comandos comuns:


ls

get arquivo.txt # Pegar arquivo do servidor

put arquivo.txt # Enviar arquivo pro servidor, podemos passar o caminho completo como "/caminho/arquivo.txt"

bye # Encerra a conexão

Podemos digitar apenas ftp no terminal e dentro dele digitar open nomeouipdoservidor.com.br

Podemos abrir o Filezilla também pra acessar um servidor FTP de forma gráfica. Abra o programa e em File vá em Connect to Server, coloque o nome ou ip do servidor, a porta (21 pra FTP padrão ou 22 para SFTP) e a senha, e clique em Ok.

Um exemplo de servidor FTP público que podemos acessar é esse descrito abaixo, com esses dados:


Host: ftp.gnu.org
Usuário: anonymous
Senha: qualquer email
Porta: 21

SSH

SSH é um acesso remoto seguro a outro computador. Da mesma forma, fazemos assim:


ssh usuario@nomeouipdoservidor.com

PS: Em algumas distros Linux ele pode não estar instalado, para instalar o cliente no Linux, procure pelo pacote openssh-client.

Pode ser assim também:


ssh -l usuario nomeouipdoservidor.com

Exemplo com porta diferente:


ssh -p 2222 usuario@servidor.com

Depois disso use os mesmos comandos do Unix para navegar. Para sair digite logout ou exit.

Um servidor que podemos usar pra testes é esse, no qual vamos avançando de nível com a conexão SSH, nesse site aqui: https://overthewire.org/wargames/bandit/

Para conectar no servidor acima, basta usar ssh bandit0@bandit.labs.overthewire.org -p 2220 e coloque a senha bandit0.

Telnet

Telnet é um protocolo pra acesso remoto sem criptografia (antigo/inseguro).


telnet nomedoservidor.com 23

Exemplo com IP:


telnet 192.168.0.10 23

Alguns dos servidores que podemos usar para testes são o telehack.com e o towel.blinkenlights.nl (no primeiro dá pra navegar por ele, o segundo mostra imagens ASCII do Star Wars). Para usar um deles basta digitar telnet telehack.com, ele não pede senha. Pra deslogar digite logout e pra sair do Telnet digite quit.

Também é usado para testar portas:


telnet google.com 80
