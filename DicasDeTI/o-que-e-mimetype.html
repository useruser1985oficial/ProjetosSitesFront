O que são MIME Types?

Um tipo de mídia de Internet ou tipo MIME é um identificador padrão usado na internet para indicar o tipo de dado que um arquivo contém, já que pra internet, pros navegadores e para alguns sistemas operacionais como o Linux, as extensões não tem importância e sim o MIME type do arquivo.

Esses são alguns dos MIME Types e os arquivos cujas extensões são equivalentes:

Para Texto:

Extensões Parâmetro
HTML text/html
CSS text/css
JS text/javascript
Outros Tipos de Texto text/plain

Para Imagens:

Extensões Parâmetro
JPG, JPEG image/jpeg
PNG image/png
GIF image/gif
BMP image/bmp
WEBP image/webp

Para áudio:

Extensões Parâmetro
MP2, MP3 audio/mpeg
WMA audio/x-ms-wma
M4A audio/x-aac
OGG audio/ogg
WAV audio/wav
MIDI audio/midi

Para vídeo:

Extensões Parâmetro
MP4 video/mp4
AVI video/avi
MPEG video/mpeg
WMV video/x-ms-wmv
WEBM video/webm
OGV video/ogg

Para aplicações:

Extensões Parâmetro
Executáveis do Linux application/x-executable
SH application/x-sh - text/x-shellscript
EXE application/vnd.microsoft.portable-executable - application/x-msdownload
MSI application/x-msi - application/octet-stream
REG application/x-ms-regedit
BAT application/x-bat
JAR application/java-archive
XML application/xml
PDF application/pdf
SWF application/x-shockwave-flash
ZIP application/zip

Para ver todos os MIME Types, podemos olhar nesse site: https://www.iana.org/assignments/media-types/media-types.xhtml
